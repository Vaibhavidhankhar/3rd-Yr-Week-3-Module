# 3rd-Yr-Week-3-Module
# Sorting Algorithms in Java

This repository contains Java implementations of three fundamental sorting algorithms: **Merge Sort**, **Quick Sort**, and **Heap Sort**. Each algorithm is implemented through **Easy** and **Medium** level real-world problem statements to demonstrate both algorithmic concepts and practical applications.

The programs are designed to strengthen understanding of sorting techniques, recursion, divide-and-conquer strategies, heap data structures, and analytical problem solving while maintaining efficient performance for large datasets.

---

## Objectives

- Understand the working of Merge Sort, Quick Sort, and Heap Sort.
- Implement efficient sorting algorithms without using built-in sorting methods.
- Apply sorting techniques to real-world scenarios.
- Perform additional statistical analysis after sorting.
- Improve understanding of recursion, partitioning, heap construction, and divide-and-conquer techniques.

---

#  Merge Sort

Merge Sort is a **Divide and Conquer** algorithm that recursively divides the array into smaller subarrays, sorts each half, and finally merges them to produce a completely sorted array.

### Programs Included

###  Question 1 (Easy)

**Scenario:**
Sort the salary package values of placed students in ascending order.

**Operations Performed**

- Read salary package values
- Sort using Merge Sort
- Display sorted salary packages

---

###  Question 2 (Medium)

**Scenario:**
Sort customer order processing times for an e-commerce company and perform statistical analysis.

**Operations Performed**

- Sort processing times
- Find median
- Count orders above median
- Calculate difference between fastest and slowest processing times

---

#  Quick Sort

Quick Sort is a **Divide and Conquer** algorithm that selects a pivot element, partitions the array around the pivot, and recursively sorts the resulting partitions.

### Programs Included

### Question 1 (Easy)

**Scenario:**
Sort product prices in ascending order using Quick Sort.

**Operations Performed**

- Read product prices
- Sort using Quick Sort
- Display sorted prices

---

### Question 2 (Medium)

**Scenario:**
Sort stock market trade values in descending order and generate analytical statistics.

**Operations Performed**

- Sort trade values in descending order
- Display Top 5 highest trade values
- Calculate Top 5 average
- Calculate overall average
- Count values greater than overall average

---

# Heap Sort

Heap Sort is based on the **Binary Heap** data structure. A Max Heap is constructed, and the largest element is repeatedly moved to its correct position to obtain the sorted array.

### Programs Included

### Question 1 (Easy)

**Scenario:**
Sort player scores collected from an online gaming tournament.

**Operations Performed**

- Read player scores
- Build Max Heap
- Sort scores in ascending order
- Display sorted leaderboard

---

### Question 2 (Medium)

**Scenario:**
Analyze emergency response times recorded by a hospital.

**Operations Performed**

- Sort response times
- Find fastest response time
- Find slowest response time
- Calculate average response time
- Count cases faster than average
- Calculate percentage of cases faster than average

# ⚡ Time Complexity

| Algorithm | Best | Average | Worst | Space |
|-----------|-------|---------|--------|--------|
| Merge Sort | O(N log N) | O(N log N) | O(N log N) | O(N) |
| Quick Sort | O(N log N) | O(N log N) | O(N²) | O(log N) |
| Heap Sort | O(N log N) | O(N log N) | O(N log N) | O(1) |
